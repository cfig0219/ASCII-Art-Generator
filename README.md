# Creative ASCII Art Generator (C++)

## 📖 Overview
This repository showcases a C++ application that utilizes algorithmic logic to render creative ASCII art in a console environment. The project demonstrates a strong command of procedural programming, nested loop structures, and conditional rendering to transform code into visual design.

## 🚀 Key Features
* **Procedural Rendering:** Employs optimized C++ logic to generate complex visual patterns.
* **Creative Logic:** Features custom ASCII designs developed through precise coordinate mapping.
* **Modular Architecture:** Organized into distinct source files (`ChatCodeHW5.cpp`, `RixCodeHW5.cpp`) to handle different rendering algorithms and pattern sets.
